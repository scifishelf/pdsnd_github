# US Bikeshare Analysis

### Date created
April 26, 2026

### Project Title
US Bikeshare Analysis

### Description
This project analyzes bikeshare data for Chicago, New York City, and Washington.
The Python script is designed to load city data, filter it by month and day, and
display summary statistics about travel times, stations, trip duration, and users.

### Requirements
- Python 3
- pandas

### How to run
Place the city CSV files in the project root, then run:

```bash
python bikeshare.py
```

### Files used
- `bikeshare.py`
- `chicago.csv`
- `new_york_city.csv`
- `washington.csv`
